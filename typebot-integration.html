Integração do Chatbot Typebot

Este documento contém o código necessário para integrar o chatbot do Typebot ao site da Digita Money.

Nota: Você precisará substituir SEU_URL_DO_TYPEBOT pela URL real do seu bot no Typebot.

Código para Adicionar ao Site

Adicione o seguinte código antes da tag de fechamento </body> em todas as páginas do site:

<!-- Typebot Integration -->
<script src="https://cdn.jsdelivr.net/npm/@typebot.io/js@0.1/dist/web.js"></script>
<script>
    document.addEventListener('DOMContentLoaded', function() {
        // Configuração do Typebot
        window.Typebot.initBubble({
            typebot: "SEU_URL_DO_TYPEBOT",
            theme: {
                button: {
                    backgroundColor: "#00E5FF",
                    iconColor: "#000000"
                },
                chat: {
                    backgroundColor: "#333333",
                    textColor: "#FFFFFF",
                    fontSize: 16
                },
                previewMessage: {
                    backgroundColor: "#00E5FF",
                    textColor: "#000000"
                }
            },
            previewMessage: {
                message: "Olá! Como posso ajudar você hoje?",
                autoShowDelay: 5000
            },
            buttonPosition: "right",
            buttonText: "Fale Conosco",
            buttonIconUrl: "images/chat-icon.svg"
        });
        
        // Integração com o botão de chat existente
        const chatbotToggle = document.querySelector('.chatbot-toggle');
        const typebotContainer = document.getElementById('typebot-container');
        
        if (chatbotToggle) {
            chatbotToggle.addEventListener('click', function() {
                window.Typebot.toggleBubble();
                chatbotToggle.classList.toggle('active');
            });
        }
    });
</script>

Modificações no CSS

Adicione o seguinte CSS ao seu arquivo style.css para estilizar o botão do chatbot quando estiver ativo:

.chatbot-toggle.active {
    background: linear-gradient(135deg, #00FF9D, #00E5FF);
    transform: scale(1.1) rotate(45deg);
}

.chatbot-toggle.active i {
    transform: rotate(-45deg);
}

Instruções de Implementação

  1. Obtenha a URL do seu bot no Typebot após a publicação
  2. Substitua SEU_URL_DO_TYPEBOT pela URL real no código acima
  3. Adicione o código antes da tag de fechamento </body> em todas as páginas
  4. Adicione o CSS ao seu arquivo de estilos
  5. Teste a integração para garantir que o chatbot está funcionando corretamente

Dica: Você pode personalizar as cores, mensagens e comportamento do chatbot editando os parâmetros no código acima.
